# 瞄點直播 Demo Landing

展示用靜態網站，包含：
- Hero 主視覺、功能介紹
- 收費方案（含模擬付款對話框）
- 登入/註冊表單（localStorage 模擬）

部署方式
- 內建 `.github/workflows/pages.yml`，推送至 `main` 後會自動部署到 GitHub Pages。
- 若 Repo 名稱為 `Cueaim`，公開網址將會是：
  `https://<你的帳號>.github.io/Cueaim/`

本機預覽
- 直接開啟 `index.html` 或使用內建開發伺服器
  http://localhost:3000/landing/index.html
